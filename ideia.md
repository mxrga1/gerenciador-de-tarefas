# Ideia do Projeto

## Nome do projeto
Gerenciador de Tarefas

## Problema
Muitas pessoas têm dificuldade em organizar tarefas do dia a dia, acompanhar atividades pendentes e manter controle sobre prazos.

## Público-alvo
- Estudantes  
- Pequenas equipes  
- Usuários que desejam organizar tarefas pessoais  

## Objetivo
Desenvolver um sistema web simples para gerenciamento de tarefas, permitindo organizar atividades de forma prática.

## Funcionalidades principais
1. Criar tarefa  
2. Visualizar tarefas  
3. Marcar tarefa como concluída  
4. Editar tarefa  
5. Excluir tarefa  
6. Definir prazo (opcional)  

## Descrição da solução
O sistema permitirá ao usuário cadastrar tarefas, visualizar lista de atividades, editar informações, excluir tarefas e marcar tarefas concluídas. Também será possível adicionar prazo opcional para melhor organização.
